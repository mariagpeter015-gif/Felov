# 📚 Book Tracker Web Application

A full-stack web application built using Django that allows users to track and manage their reading progress efficiently. The application enables users to add books, update their reading status, and view completed books through a simple and user-friendly dashboard.

---

## 🚀 Features
- Add and manage multiple books
- Mark books as completed
- View completed books on a dashboard
- Clean and intuitive user interface

---

## 🛠 Tech Stack
- **Backend:** Django, Python  
- **Frontend:** HTML, CSS  
- **Database:** SQLite (default Django database)

---

## ▶️ How to Run the Project

### 1. Clone the repository
### 2. Create a virtual environment (optional but recommended)
### 3. Install dependencies
### 4. Apply migrations
### 5. Run the development server
### 6. Open in browser
